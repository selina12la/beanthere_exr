# BeanThere – EXR Project

## 📌 Project Overview
BeanThere is a Unity-based VR experience developed as part of the EXR project.

## 🛠 Built With
- Unity (version: 6000.3.10f1)
- XR Interaction Toolkit
- Target platform: (Meta Quest Pro.)

## 👥 Team
- Selina Lanz
- Michelle Rickenmann
- Simy Melinda Tran

## 📂 Project Structure
- Assets/ → Game assets and scripts
- Packages/ → Unity packages
- ProjectSettings/ → Unity configuration files

## 🚀 Setup Instructions
1. Clone the repository
2. Open the project using Unity Hub
3. Make sure the correct Unity version is installed

## ⚙️ Git Rules
- Visible Meta Files enabled
- Asset Serialization set to Force Text
- Do NOT commit Library/ or Temp/ folders
